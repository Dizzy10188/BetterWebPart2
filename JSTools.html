To the left is a Tab bar. You can have as many tabs as you want, but width will be a factor in it. To make this Tab bar, use the code shown below.
<div class="bg-black"> <div class="bar-item" onclick="openTab('Link 1')">Link 1</div> <div class="bar-item" onclick="openTab('Link 2')">Link 2</div> <div class="bar-item" onclick="openTab('Link 3')">Link 3</div> </div> <div id="Link 1" class="bar-tabs tab-container"></div> <div id="Link 2" class="bar-tabs tab-container"></div> <div id="Link 3" class="bar-tabs tab-container"></div>
