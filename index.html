To the left is a dropdown box. The way to create a dropdown box is by using the code below

Header

Nav-bar

On the top of the page is a Nav-bar. The way to create a Nav-bar is by using the code below
<nav class="sticky-nav"> <ul> <li><a href="#">Link 1</a></li> <li><a href="#">Link 2</a></li> <li><a href="#">Link 3</a></li> <li><a href="#">Link 4</a></li> </ul> </nav>

Boxes

To the left are 4 boxes, each different colors. The way to create these bozes are by using the code below. The box's size changes by multiples of 10, so box-green-100 works as well
<div class="box-green-50"></div> <div class="box-red-50"></div> <div class="box-silver-50"></div> <div class="box-black-50"></div> <div class="box-blue-50"></div> <div class="box-cyan-50"></div>
